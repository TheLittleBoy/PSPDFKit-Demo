itemWithType:variant:configurationBlock:

Creates a group item with the specified annotation type, an optional variant identifier and configuration block.

+ (instancetype)itemWithType:(NSString *)type variant:(NSString *)variant configurationBlock:(PSPDFAnnotationGroupItemConfigurationBlock)block

Parameters

type

The annotation type. See PSPDFAnnotation.h for a list of valid types.

variant

An optional string identifier for the item variant. Use variants to add several instances of the same tool with uniquely preservable annotation style settings.

block

An option block, that should return the button’s image. Defaults to a block that configures an preset image based on the annotation type, if set to nil.
