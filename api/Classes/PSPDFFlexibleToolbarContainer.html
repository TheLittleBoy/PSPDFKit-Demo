Inherits from UIView
Declared in PSPDFFlexibleToolbarContainer.h

Overview

The flexible toolbar container holds and manages a PSPDFFlexibleToolbar instance. Its main responsibilities include toolbar anchoring and drag & drop handling. Add this view to your view hierarchy (a good candidate might be the UINavigationController’s view).

hideAndRemoveAnimated:completion:

Hides the container and toolbar using hideAnimated:completion: and than removes the container from it’s superview. Internally used by the flick to close gesture. Should also be used by done / close buttons added to the toolbar.
